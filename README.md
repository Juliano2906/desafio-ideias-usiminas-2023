# desafio-ideias-usiminas-2023
Participação no Desafio de Ideias +3i do SENAI (out/2023). Em equipe, resolvemos demanda da USIMINAS: aumentar rendimento térmico da caldeira, linha de distribuição e trocadores de calor, reduzindo consumo de Gás Natural e perdas energéticas. Solução: otimização térmica + recuperação de energia. Resultado: 1º lugar Regional.
